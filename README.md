## Deployment to GitHub Pages

This landing page is ready to be hosted as a **single-page application** via GitHub Pages.

1.  Push the current contents of this directory to your GitHub repository (e.g., `github.com/vumarix/vumarix`).
2.  Enable GitHub Pages in the repository settings (referencing the main branch).
3.  The `CNAME` file is already included for `vumarix.com`.
